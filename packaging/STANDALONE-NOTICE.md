# Standalone Distribution Notice

The standalone Linux executables and the system packages built from them
embed a Node.js runtime in order to run without a separately installed Node.js
package.

The project source is distributed under the MIT License in `LICENSE`.
The license and third-party notices applicable to the embedded Node.js runtime
are distributed alongside the executable as `NODE-LICENSE`, and are installed
under `/usr/share/doc/codex-history-viewer/` by system packages.
